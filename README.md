# County Referral Hospital & Pharmacy Analytics Dashboard | Power BI Project
(Professional, clear, and attractive to recruiters.)

## Project Overview
This project analyzes hospital patient and pharmacy data to generate actionable insights for a county referral hospital. The goal is to support management in understanding patient visit trends, disease patterns, pharmacy revenue performance, and medication consumption across departments and counties.
